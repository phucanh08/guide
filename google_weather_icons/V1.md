# API

https://ssl.gstatic.com/onebox/weather/{`size`}/{`weather-condition`}.png

- `size` là:
    - 32
    - 48
    - 64
    - 96
    - 128
    - 256

- `weather-condition` là: 
    - sunny
    - sunny_s_rain
    - sunny_s_snow
    - sunny_s_cloudy
    - cloudy
    - partly_cloudy
    - cloudy_s_rain
    - cloudy_s_snow
    - cloudy_s_sunny
    - rain
    - rain_light
    - rain_heavy
    - rain_s_cloudy
    - rain_s_sunny
    - rain_s_snow
    - snow
    - snow_light
    - snow_heavy
    - snow_s_rain
    - snow_s_sunny
    - snow_s_cloudy
    - hot
    - fog
    - mist
    - sleet
    - windy
    - thunderstorms

# Ví dụ
```
    https://ssl.gstatic.com/onebox/weather/48/thunderstorms.png
    https://ssl.gstatic.com/onebox/weather/256/windy.png
    https://ssl.gstatic.com/onebox/weather/48/hot.png
    ---
```
